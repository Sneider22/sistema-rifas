## Características Principales e Innovaciones

### ⚙️ Módulo Centralizado de Notificaciones y CRM (WhatsApp Automation)
El sistema elimina la gestión manual de cobranza mediante un motor de comunicación integrado:
- **Ciclo de Estados por Cliente:** Envío automatizado de alertas personalizadas según la fase de compra (Recordatorio de Pago Pendiente ➔ Confirmación de Recibo Procesado).
- **Generador Gráfico de Tickets:** Al concretarse el pago, la plataforma renderiza de forma nativa un **ticket digital con imagen personalizada**, listo para ser compartido.
- **Integración Directa con WhatsApp:** Despacho rápido de comprobantes y plantillas de texto optimizadas directamente al chat del participante.

### 🗂️ Panel Maestro de Tickets
Un centro de control exclusivo para la auditoría de fracciones:
- **Gestión de Estados Instantánea:** Permite conmutar números a "Pagado" con un solo clic.
- **Portapapeles Avanzado:** Funciones rápidas para copiar la data del ticket en texto plano o previsualizar el diseño gráfico del boleto emitido.

### 👥 Base de Datos de Jugadores
- Historial exhaustivo de compras indexado por participante.
- Filtros rápidos para reconocer la frecuencia de juego y transacciones acumuladas.

### 🛠️ Configuración Global de Empresa
Panel administrativo avanzado para moldear la identidad del negocio:
- **Personalización de Marca:** Ajustes del perfil comercial y datos de contacto de la organización.
- **Pasarelas y Métratos de Pago:** Configuración dinámica de cuentas, bancos y opciones de pago móvil/transferencias.
- **Configurador de Plantillas:** Sistema de macros para precargar los mensajes predeterminados que se dispararán por WhatsApp.

## Módulos del Sistema

| Módulo | Descripción Funcional |
| :--- | :--- |
| **Dashboard** | Resumen analítico instantáneo, ingresos brutos y Top 5 de compradores del mes. |
| **Rifas** | Catálogo de sorteos activos y finalizados con flujos de comunicación integrados. |
| **Tickets Panel** | Grilla administrativa para copiar, marcar estados y previsualizar imágenes de boletos. |
| **Jugadores** | Directorio integral de clientes con métricas individuales de consumo. |
| **Estadísticas** | Gráficos temporales de conversión, eficiencia de ventas y distribución porcentual. |
| **Configuración** | Gestión de mensajes tipo, pasarelas de pago y datos maestros de la firma. |
